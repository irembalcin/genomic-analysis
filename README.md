# Genomic Analysis: GC Content & ORF Detection

This project demonstrates basic DNA sequence analysis using Python and Biopython.

## Features

- Load and parse FASTA format
- Calculate GC content
- Detect Open Reading Frames (ORFs)
- Visualize GC content with sliding window
